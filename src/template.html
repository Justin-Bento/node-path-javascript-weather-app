Location Information:

  • City Name (e.g., "London")
  • Country (e.g., "UK")
  • Latitude/Longitude (for more accurate location-based weather)

Current Weather Conditions:

  • Temperature (in Celsius, Fahrenheit, or Kelvin)
  • Feels Like (the "real feel" temperature, factoring in humidity and wind)
  • Weather Description (e.g., "Clear", "Partly Cloudy", "Rainy")
  • Weather Icon (URL or code for showing an appropriate icon, like a sun, cloud, or raindrop)
  • Humidity (percentage)
  • Wind Speed (in km/h, mph, or m/s)
  • Wind Direction (degrees or cardinal direction, e.g., "N", "NE")
  • Precipitation (Rain, Snow, etc.)
  • Pressure (Atmospheric pressure in hPa, millibars, or inches of mercury)
  • UV Index (to measure the intensity of UV radiation)
  • Visibility (distance in meters or miles)
